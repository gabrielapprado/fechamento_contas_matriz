
# 🍕 Franzinnis Pizzeria Billing System

## About the Project

This project was developed as a practical assignment for the Logic and Programming course at FATEC. Its main goal was to apply fundamental programming concepts in C, including arrays, matrices, conditional statements, loops, data validation, and console interface manipulation.

The application simulates the billing process of a pizzeria, allowing the registration of table orders, automatic calculation of service charges and discounts, bill splitting among customers, and generation of a daily revenue report.

The interface was built using a DOS-style console layout with the gotoxy function, providing a structured text-based user experience.

## Features

* Register orders for up to 10 tables
* Track pizzas, toppings, draft beers, soft drinks, and water
* Automatic service charge calculation
* Progressive discount system
* Bill splitting per customer
* Revenue accumulation by table
* Daily revenue report
* Average revenue calculation per table

## Concepts Practiced

* One-dimensional arrays
* Two-dimensional arrays (matrices)
* Conditional statements
* Loop structures
* Functions
* Data validation
* Console cursor positioning
* Real-world business logic implementation

## Technologies

* C Language
* DOS-style console interface
* Gotoxy-based screen navigation

## Educational Purpose

This project was created to strengthen logical reasoning and programming fundamentals through the development of a real-world restaurant billing simulation. It was proposed as part of the Logic and Programming course at FATEC.
